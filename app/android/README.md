# App Directory

Este diretório contém os binários do aplicativo para testes.

## Android

Baixe o APK mais recente do repositório oficial:

```
https://github.com/webdriverio/native-demo-app/releases
```

Salve como: `android-wdio-demo-app.apk`

## iOS

Baixe o app para simulator:

```
https://github.com/webdriverio/native-demo-app/releases
```

Descompacte e salve em: `ios-wdio-demo-app.app`

## Estrutura

```
app/
├── android/
│   └── android-wdio-demo-app.apk
└── ios/
    └── ios-wdio-demo-app.app/
```

**Nota**: Os arquivos de app não estão incluídos no repositório por questões de tamanho. Baixe-os separadamente.
